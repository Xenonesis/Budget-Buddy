### ğŸ› ï¸ Installation Steps

#### 1. **Clone the Repository**
```bash
git clone https://github.com/Xenonesis/Budget-Buddy.git
cd Budget-Buddy
```

#### 2. **Install Dependencies**
```bash
npm install
# or
yarn install
```

#### 3. **Environment Setup**
Copy the environment template:
```bash
cp .env.template .env.local
```

Configure your `.env.local` file with your credentials:
```env
# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

# AI Configuration (Optional - for AI features)
NEXT_PUBLIC_GOOGLE_AI_API_KEY=your_google_ai_api_key
NEXT_PUBLIC_CLAUDE_API_KEY=your_claude_api_key
NEXT_PUBLIC_MISTRAL_API_KEY=your_mistral_api_key

# App Configuration
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

#### 4. **Database Setup**
Run the database setup scripts in your Supabase SQL Editor:
```sql
-- Execute these files in order:
sql/setup-1-base.sql
sql/setup-2-security.sql
sql/setup-3-functions.sql
sql/setup-ai-tables.sql
```

#### 5. **Start Development Server**
```bash
npm run dev
# or
yarn dev
```

#### 6. **Open Your Browser**
Visit [http://localhost:3000](http://localhost:3000) to see your application! ğŸ‰

## ğŸ”§ API Reference

Budget Buddy provides a comprehensive REST API for programmatic access to all features:

### ğŸ” Authentication

All API requests require authentication via JWT tokens obtained through Supabase Auth.

```http
Authorization: Bearer YOUR_ACCESS_TOKEN
Content-Type: application/json
```

### ğŸ“Š Transactions API

#### Get All Transactions
```http
GET /api/transactions
```

**Query Parameters:**
- `limit` (integer) - Number of transactions to return (default: 50)
- `offset` (integer) - Offset for pagination (default: 0)
- `category` (string) - Filter by category
- `startDate` (date) - Filter by start date
- `endDate` (date) - Filter by end date

**Response:**
```json
{
  "transactions": [
    {
      "id": "uuid",
      "amount": 125.99,
      "description": "Grocery shopping",
      "category": "Food & Dining",
      "date": "2024-08-15T14:30:00Z",
      "type": "expense",
      "account": "Checking",
      "createdAt": "2024-08-15T14:30:00Z",
      "updatedAt": "2024-08-15T14:30:00Z"
    }
  ],
  "total": 1,
  "limit": 50,
  "offset": 0
}
```

#### Create Transaction
```http
POST /api/transactions
```

**Request Body:**
```json
{
  "amount": 125.99,
  "description": "Grocery shopping",
  "category": "Food & Dining",
  "date": "2024-08-15T14:30:00Z",
```

```http

```

#### Delete Transaction
```http

```
